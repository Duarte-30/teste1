Links e WWW

A importância dos hyperlinks na World Wide Web (www)

O texto fala sobre a importância dos hyperlinks na World Wide Web e como criar e configurar âncoras para vários comportamentos. Os hyperlinks são a essência fundamental do hipertexto, permitindo que documentos sejam conectados entre si e a navegação seja possibilitada entre essas páginas.

Para criar um hyperlink, é necessário criar âncoras usando a tag a e definir o atributo href que cria uma referência hipertexto para outro site. O atributo hreflang é bem útil, permitindo indicar o idioma principal do site para onde o link está desviando o fluxo de navegação.

Por padrão, os sites de destino são abertos na mesma janela do site que continha o link, mas para controlar onde o site de destino vai abrir, é possível usar o atributo target com valores como _blank, _self, _top, _parent e nome-do-frame.

Também é possível indicar a natureza do destino usando o atributo rel que aceita valores como next, prev, author, external e nofollow.

O texto ainda aborda a importância dos hyperlinks para os mecanismos de busca, que vasculham constantemente todos os outros sites da Internet procurando por links para descobrir novos conteúdos. É por isso que é tão importante conseguir links válidos de outros sites para o nosso próprio site.

O autor ainda oferece exemplos de como criar links internos e externos e como usar os atributos href, hreflang, target e rel.

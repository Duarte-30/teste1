Falando sobre criar um site de noticias

O texto fala sobre a criação de um projeto simples de site de notícias, com o objetivo de ensinar conceitos adicionais para torná-lo real. O primeiro passo é planejar o layout do site utilizando um wireframe, que servirá de base para planejar as caixas que farão parte da página. Para isso, é recomendado utilizar a ferramenta MockFlow, que permite planejar layouts para sites, aplicativos desktop e até aplicativos de celular.

Em seguida, é importante decidir a paleta de cores que será utilizada no site. É recomendado utilizar uma técnica de monocromia a partir de tons de verde. Para escolher as cores, pode-se utilizar a ferramenta Adobe Color, que foi ensinada no capítulo 13. As fontes escolhidas foram a Bebas Neue para os destaques principais, a iDroid para a matéria sobre Android e a Arial para o texto padrão.

O texto também aborda o conceito de responsividade, que consiste na capacidade do site de se adaptar a diferentes dispositivos e tamanhos de tela. Para isso, é importante utilizar técnicas como largura e altura adaptáveis, indicando valores mínimos e máximos para as caixas. No caso do projeto proposto, é recomendado definir os limites da largura da caixa principal do conteúdo em 400px e 800px, para garantir que ela não fique com largura menor que 400px em telas pequenas e nem seja maior que 800px em telas muito grandes.

Além disso, o texto ensina como utilizar variáveis personalizadas com CSS para cadastrar esquemas de cores e fontes no site. Para isso, deve-se definir uma área de definições dentro do estilo para uma pseudo-classe chamada :root, que definem as configurações para a raiz de uma árvore. Isso permite facilitar mudanças estéticas no site no futuro.

Por fim, o texto destaca a importância de aprender conceitos adicionais como as media queries com a regra @media e as caixas flexíveis com as flex boxes para dar ainda mais capacidade de adaptação do conteúdo do site.
